# 3D Portfolio

A modern, interactive 3D portfolio website built with Next.js, React Three Fiber, and Tailwind CSS.

## Features

- **3D Elements** — Immersive 3D visuals using React Three Fiber and Drei
- **Smooth Scrolling** — Fluid scroll experience with Lenis
- **Animations** — Engaging animations powered by Framer Motion and GSAP
- **Responsive Design** — Fully responsive across all devices
- **Modern Stack** — Built with Next.js 16, React 19, and Tailwind CSS 4

## Tech Stack

- Next.js 16
- React 19
- Three.js / React Three Fiber / Drei
- Tailwind CSS 4
- Framer Motion
- GSAP
- Lenis (smooth scrolling)
- Lucide React (icons)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
